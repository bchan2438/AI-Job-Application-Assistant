# AI-Job-Application-Assistant
This assistant is designed to significantly speed up job applications
<br><br>
The assistant will be set up to work with the following websites: <br>Linkedin<br>Indeed<br>Ziprecruiter
<br><br> 
Given search parameters, the assistant will return job details:<br>title<br>Pay<br>Location<br>Company Name<br><br>A score from the ai about how relevant the job is to your skills(way of inputting these tbd, but resume + txt or pdf files should be supported)<br>
If the score is above X/100, the assistant will also generate a cover letter modelled after a cover letter you upload(pdf most likely)<br><br>

All of this data will be outputted in a .csv file(not sure how i will get the cover letters to work quite yet, this is TBD)
